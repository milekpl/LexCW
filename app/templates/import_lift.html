Import Guidelines
Supported File Format
  • LIFT (Lexicon Interchange FormaT) XML files
  • Files with .lift or .xml extensions
  • UTF-8 encoded files are recommended
  • .lift-ranges files for semantic domains and grammatical information (highly recommended)
Entry Structure
  • Each entry must have a unique ID or one will be generated
  • Lexical units (headwords) are required
  • Senses, definitions, and examples are optional but recommended
  • Pronunciation data will be preserved if present
Import Process
  • The file will be validated for proper XML structure
  • Entries will be parsed and validated individually
  • Import statistics will be displayed upon completion
  • Any errors or warnings will be logged and reported
