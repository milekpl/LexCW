Import FieldWorks list.xml

Upload a list.xml file exported from FieldWorks Language Explorer (File > Export > List of Lists). This provides real abbreviations (e.g., "irreg. infl.", "comp.", "der.") for Variant Types, Complex Form Types, Parts of Speech, and other controlled vocabularies.

What is list.xml?

FieldWorks Language Explorer (FLEx) stores controlled vocabularies (Variant Types, Complex Form Types, Parts of Speech, etc.) in list.xml. This file contains the real abbreviations and labels that your dictionary project uses.

How to export list.xml from FLEx
  1. In FLEx, go to File > Export > List of Lists
  2. Save the output as list.xml
  3. Upload it using the form above
What gets imported
  • Variant Types — abbreviations like "irreg. infl.", "comp."
  • Complex Form Types — abbreviations for compound/derivative types
  • Parts of Speech — POS labels and abbreviations
  • Semantic Domains — domain classification labels
  • Other FieldWorks possibility lists that map to app ranges
When to use this

Use this page if you have already imported your LIFT file but want to add real FieldWorks abbreviations afterwards. You can also import list.xml during the initial LIFT import on the Import LIFT page.
